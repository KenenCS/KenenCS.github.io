GitPages博客开通详解

    费了一点时间通过GitHub Pages开通了自己的博客,这个博客我准备向大家分享自己作为程序员一路走来的点点滴滴,有空我就会更新,专业技术性的文章会较多一点,个人感觉一个好的程序员要懂得分享,只有在分享讲解中才能让自己成长的快些,给别人讲的同时说明自己懂了,这是利人利己的事儿!我的邮箱是KenenCS@163.com,欢迎多多交流!

MAC系统下GitHub Pages博客搭建步骤:

一.常见的博客搭建方案分析

WorkPress：一般需要独立域名（收费），对 MarkDown 不友好
GitHubPages + Jekell：免费，稍微有点麻烦
GitHubPages + Hexo：免费，使用简单，使用者众多…
Hexo 是一个基于 Nodejs 的静态博客网站生成器,速度很不错,支持 Markdown,兼容于 Windows, Mac & Linux.

二.博客环境准备

1.创建GitHub仓库
①登录GitHub账号(没有账号不会注册的就看这儿);
②登录成功后创建一个新的仓库,注意Respository name命名格式一定是”你的用户名.github.io”,如KenenCS.github.io(不会创建仓库的看这儿);
2.本地环境准备
①下载安装 Git.
    Mac中的Xcode已经自带的有Git不用执行这一步,但没装Xcode的则要进行这一步,首先打开 Git官网,选择 Downloads 下面的 Mac OS X,下载完成后直接双击 dmg 文件安装即可.
②下载安装 Nodejs.
    打开 Nodejs 下载页面，选择 Macintosh Installer 即可开始下载,下载完成后直接双击 pkg 文件安装即可.
③下载安装 Hexo.
    可以先看下 Hexo官网.
    打开终端,输入命令$ npm install hexo-cli -g

三.创建博客

1.创建本地博客
    ①打开终端,输入命令$  cd 你的博客文件夹
    ②输入命令$  hexo init 你的用户名.github.io  (这里就是前面提到的命名注意)
2.配置博客信息
    首先，复制一份打开本地博客目录下的 _config.yml 文件，命名为 _config_bak.yml，做为备份，以防改错…然后使用文本编辑器打开本地博客目录下的 _config.yml 文件.
    这里先看看最基础的配置，也就是必须要改动的(注意,键值之间一定要加空格)：

title: xxx # 博客的名字，也称站点名称
author: xxx # 作者名字
description: xxx # 对站点的描述，搜索引擎会抓取，可以自定义（这个还是加上比较好）
language: zh-Hans # 语言 简体中文
theme: next # 配置主题
deploy: # 部署相关配置
type: git # 使用 Git 提交
repo: https://github.com/xxx/xxx.github.io.git # 就是存放博客的仓库地址
    具体请看 Hexo文档.
3.查看自己的博客
    ①打开终端,输入命令行$  cd xxx.github.io
    ②输入命令$  hexo s
    ③运行成功后,有一个网址 http://localhost:4000/ 打开它,会看到一篇Hello World.然后回到终端, control+c 关闭.
    ④安装自动部署发布工具,输入命令$  npm install hexo-deployer-git –save
    ⑤刚你看到的 http://localhost:4000/ 页面其实是调试的状态下,并没有发布,现在发布,输入命令行$  hexo clean && hexo g && hexo d ,发布完成,在浏览器上输入 xxx.github.io 查看你的博客吧!
4.写博客与发布
    ①打开你本地博客的文件夹,在xxx.github.io/source/_posts 文件夹下面你会看到Hello World,打开这个文件之后你就会发现这个就是你的第一篇博客源码.
    ②打开终端,输入命令行$  cd 你的博客文件夹
    ③打开终端,输入命令行$  cd xxx.github.io
    ④创建新文章,输入命令行$  hexo new”postName” ,你会发现在你的Hello World源码下有多了一个新文件,这个就是你刚建的新文件,然后你在按照第2步(配置博客信息)和第3步(查看自己的博客)走一遍,就新建了一篇文章.

四.更换博客主题

如果感觉你的博客主题不好看,那么咱可以换!目前网上有很多 Hexo 主题.

1.打开终端,输入命令$  cd 你的博客文件夹
2.输入命令$  git clone https://github.com/iissnan/hexo-theme-next themes/next (https://github.com/iissnan/hexo-theme-next这个链接就是你找的主题的GitHub链接).
3.使用文本编辑器打开本地博客目录下的 _config.yml 文件，搜索，定位 theme 键值，将 theme 的值修改为 next.(注意哦,_config.yml 还是先复制一份留作备份)
4.输入命令$  hexo s 进行查看页面主题合不合自己口味.
5.输入命令行$  hexo clean && hexo g && hexo d ,进行发布!



以上就是创建博客的流程,如果发现有报错的话可以联系我,或者可以进入 Hexo 官网查看错误总结!
